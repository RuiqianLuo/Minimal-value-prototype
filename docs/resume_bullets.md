# Resume Bullets For AI Product Roles

- Built an internal knowledge assistant prototype using a Python RAG workflow that ingests company documents, retrieves grounded evidence, and generates cited answers through a lightweight web app.
- Designed an evaluation framework for usefulness, faithfulness, hallucination rate, latency, and retrieval hit quality using benchmark questions, expected sources, and structured run logs.
- Authored a product case study covering PRD, user personas, MVP tradeoffs, success metrics, and error analysis to demonstrate product thinking alongside technical execution.
- Scoped the system as an enterprise productivity use case with clear MVP boundaries and future roadmap decisions around retrieval quality, cost, speed, and trust.
