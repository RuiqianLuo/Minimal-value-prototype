# Feature Prioritization

## MVP

- Local document ingestion
- Chunking and retrieval pipeline
- Cited answer generation
- Simple internal web interface
- Evaluation benchmark with 10 questions
- Structured logs for app and eval runs
- Product documentation and error analysis

## Later

- Semantic embeddings and reranking
- Admin interface for document uploads
- User feedback collection on answers
- Access control by team or document sensitivity
- Analytics dashboard for unanswered questions and retrieval misses
- Multi-turn conversations with session memory
- Integrations with Slack, Confluence, or Google Drive
