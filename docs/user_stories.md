# User Stories

- As a new employee, I want to ask policy questions in plain language so I can find answers without knowing document names.
- As a manager, I want source citations in each answer so I can verify guidance before acting on it.
- As an operations lead, I want fewer repetitive questions routed to shared channels so the team saves time.
- As an AI product analyst, I want benchmark questions and evaluation logs so I can assess answer quality over time.
- As a product manager, I want clear failure cases so I can decide whether the product is ready for broader rollout.
